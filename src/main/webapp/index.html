index,default 이름의 html,jsp 파일은 메인 페이지 역할을 한다

        이름과 나이를 입력받아서 서버(java 코드)로 제출

        준비물 
        form 태그 : 내부에 작성된 input값을 지정된 경로로 제출
        -action : 제출경로
        -method : 제출방법(get/post)
        제출 == 클라이언트 요청

        input 태그 : 값을 입력하는 양식제공
        - name : 서버로 제출 시 key역할
        - 입력된 값 : 서버로 제출 시 value 역할

        
        GET : get을 통한 요청은 URL 주소 끝에 파라미터로 포함되어 전송되며,
        이 부분을 query string이라 부른다
        방식 : url 끝에 ?붙이고 그 다음에 변수명1=값1&변수명2=값2&...
        ex) www.example.com/show?name1=value1&name2=value2
        -> 서버에서는 name1,name2 라는 파라미터 명으로 각각 value1,value2 
        파라미터 값을 전달받을 수 있다
        
        [GET 장점]
        -단순함
        -캐싱(저장) 가능
        	-> 주소 저장 == 북마크,즐겨찾기,주소공유
        	
        [단점]	
        - 글자수 제한이 있다
        - 보안에 취약하다(비밀번호가 주소에 보임)
